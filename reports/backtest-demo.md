## Setup

- Bars: **600** synthetic minute bars
- Price process: `100 + 10*sin(2πt/120) + 0.05*t + noise` (deterministic PRNG, seed=42)
- Strategy: 10/30 SMA crossover, long-only, fixed size 50, flat at end
- Initial cash: 100,000

**Purpose of this demo.** Exercise every part of the backtest engine
(BarCursor → strategy → execution model → equity tracking → metrics)
on a complete run. This is **not** a claim that MA crossover has edge.
The value is reproducing a full set of numbers that moves in the expected
direction when slippage/commission are introduced.
